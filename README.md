IDEOM
==========
plugin for JetBRAINS IDEs.

## Description
Customizing the JetBRAINS IDEs

## Features
* Adding background image in Editors
* Change icons at Editor
* Plays sound at event(Run, Run Error, Open Project, Close Project)
